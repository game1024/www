# AgentSkill规范
最近正在学习Skill的编写，但是我目前遇到的最大疑惑是，无法确保编写的Skill是否可靠，输入输出是否稳定，以及后续是否方便维护

## 目录结构
一个符合规范的 AgentSkill 目录结构如下
:::filetree


- SKILL.md
- scripts（命令脚本）
    - *.js
    - *.py
    - *.sh
- references（参考文档）
    - *

- assets（图片，配置，数据）
    - *
:::

## 元数据

```markdown{title="SKILL.md"}
---
name: skill-name
description: skill description
version: 1.0.0
author: author-name
license: MIT
---
```

## skill-creator
skill-creator 是 Anthropic 推出的官方的技能生成器，可以快速生成符合规范的技能，极大地降低了技能开发的门槛，推荐使用这个工具来创建技能
